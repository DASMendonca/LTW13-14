Listagem das funcionalidades implementadas

  • Base de dados onde estão armazenados os produtos, clientes e faturas utilizados nesta fase do trabalho. A base de dados poderá ser acedida por utilizadores com 3 níveis de permissão (leitura, leitura/escrita e leitura/ escrita/ gestão de utilizadores). Nesta primeira entrega apenas foram implementados as funções de leitura.
  • API de consulta, contendo, os seguintes métodos:
    • /api/searchInvoicesByField.php
    • /api/searchCustomersByField.php
    • /api/searchProductsByField.php
    • /api/getInvoice.php
    • /api/getCustomer.php
    • /api/getProduct.php
  • Pesquisa e listagem de documentos sumarizados (faturas, clientes e produtos)
  • Visualização de documentos em detalhe (um documento, por página web, com todos os detalhes relevantes)
  • Formatação de uma fatura para impressão em A4 (Actualizado)
  • API de manipulação dos documentos contendo os seguintes métodos:
    • /api/updateInvoice.php
    • /api/updateCustomer.php
    • /api/updateProduct.php
  • Edição/Criação de documentos
  • Criação de utilizadores
  • Autenticação de utilizadores
  • Gestão de utilizadores por parte dos administradores
  • Exportação/Importação de documentos no formato saft-pt[2]
  • Importação de outras BD semelhantes usando a API definida
